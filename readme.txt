# A macro pad

Hardware: Pi Pico, 9 keys in a 3 x 3 matrix, with diodes to prevent ghosting.

Installation:

* Install CircuitPython: download latest release [https://circuitpython.org/board/raspberry_pi_pico], and copy the .uf2 to the Pico

* Install hid library: download bundle [https://circuitpython.org/libraries] and copy lib/adafruit_hid to the same path on the Pico

* Copy code.py from here to the Pico
